使用return false报错问题

发表于 2021-01-27
bug报告-JavaScript

使用return false报错 Uncaught SyntaxError: Illegal return statement

问题：
1.bug

if (!window.localStorage) {
  alert("浏览器不支持localstorage");
  return false;
} else {
    console.log(123);
}

解决办法：

(function () {
  if (!window.localStorage) {
    alert("浏览器不支持localstorage");
    return false;
  } else {
    console.log(123);
  }
})()

原因：在JavaScript中return只能在函数中使用
